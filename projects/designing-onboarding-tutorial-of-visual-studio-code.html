Designing Onboarding Tutorial of Visual Studio Code

Overview

This project is to design an onboarding tutorial of Visual Studio Code (VSC), an integrated development environment (IDE) by MicroSoft, to help people who are new to VSC and coding itself feel comfortable start using VSC. It is an group project in User Experience Design class taught by Regine Gilbert at NYU in 2020 Fall semester in collaboration with a staff from MicroSoft.

Problem

Beginners have trouble with (1) how to use Visual Studio Code, and (2) write and run their first code with Visual Studio Code.

Solution

We are suggesting an onboarding tutorial. It is specifically designed for beginners who are new to Visual Studio Code and coding itself. The purpose is to guide them through their first step of coding with Visual Studio Code.
